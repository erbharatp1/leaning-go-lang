# Leaning Go Lang

A simple Go project for calculating future investment values based on user input.

## Features

- Input investment amount, expected return rate, and duration.
- Calculate future investment value using the compound interest formula.

## How to run

1. Ensure you have Go installed on your machine (you can use [mise](https://mise.jdx.dev/) to manage your tools).
2. Clone the repository.
3. Run the following command in the project root:
   ```bash
   go run main.go
   ```
4. Follow the prompts to enter your investment details.
